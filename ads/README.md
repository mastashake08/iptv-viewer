# Custom Ad Videos

Place your ad video files in this directory.

## Supported Formats
- MP4 (H.264 + AAC)
- WebM
- HLS segments (.ts files)

## Recommended Specs
- Resolution: 1280x720 or 1920x1080
- Codec: H.264 (video) + AAC (audio)
- Duration: 15-30 seconds per ad
- Bitrate: 2-5 Mbps

## Example Files
- my-ad-1.mp4
- my-ad-2.mp4
- promotional-video.mp4

These will be automatically served from `/ads/filename.mp4`
